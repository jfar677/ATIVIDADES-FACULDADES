# Senai_LLP_Romanos

Conversor de números romanos para inteiros em Java.

## Símbolos Suportados
- **I:** 1
- **V:** 5
- **X:** 10
- **L:** 50
- **C:** 100
- **D:** 500
- **M:** 1000

## Como Executar

### Passos
1. Compile o arquivo fonte:
   ```bash
   javac src/RomanoParaInt.java
   ```
2. Execute passando o algarismo como argumento:
   ```bash
   java -cp src RomanoParaInt XIV
   ```
3. Ou execute no modo interativo:
   ```bash
   java -cp src RomanoParaInt
   ```
   E digite o algarismo quando solicitado:
   `informe o algarismo: LVIII`
